Moment tensor entries represented in spherical coordinates (r-t-p (up-south-east) basis). Six distinct moment tensor entries are used to form 3 x 3 symmetric tensor in spherical coordinates. These values are used to calculate Strike / Dip / Rake for closest double couple. Strike, dip, and rake are used to describe earthquake Focal Mechanisms. These values are computed based on moment tensor entries in r-t-p (up-south-east) basis The code used for 3D moment tensor is based on Carl Tape's work. See http://www.giseis.alaska.edu/input/carl/ for papers and reference to Mathematica scripts.
